## Setup

> - Install Rust.

```sh
curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

> - Run the following scripts.

```sh
# Build the program and generate binary.
make build

# Run the binary.
make run
```
